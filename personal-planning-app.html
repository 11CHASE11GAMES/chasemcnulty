Personal Planning App

A group UX design project for a personal planning app that helps users manage tasks and stay engaged through a virtual pet bird feature.

Context

This project came from this class and was completed as a group design project.

Role

I contributed to the UX planning, interface ideas, and overall product direction.

Focus

Task planning, motivation, interaction design, and user engagement.

Problem

Planning apps can feel boring or easy to ignore. Our goal was to design a planning experience that helps users stay organized while also making the app feel more interactive.

Role

I worked with my group to shape the app concept, think through user needs, and contribute ideas for how the planning and pet features could work together.

Process

  • Identified the main goal of helping users manage tasks and schedules.
  • Planned an interactive pet bird feature to make the app more engaging.
  • Discussed how the planning tools and reward system should connect.
  • Created draft interface ideas that supported both usefulness and motivation.

Outcome

The final concept combined a practical planning tool with a simple interactive feature. This project is relevant to my portfolio because it shows my interest in UX design, user motivation, and creating digital experiences that feel useful and approachable.
